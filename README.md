## How It Works

1. GitHub Actions runs `.github/workflows/daily-post.yml` at 6:00 AM UTC
2. `generate_post.py` calls the DeepSeek API and generates one post for each of the 15 topics
3. Posts are committed to `content/{topic}/day-NN.md`
4. Quartz rebuilds and deploys the full site to GitHub Pages

## Local Usage

```bash
# Generate next day's posts for all 15 topics
python3 generate_post.py

# Generate a specific topic only
python3 generate_post.py --topic lfcs

# Generate a specific topic and day
python3 generate_post.py --topic lfcs --day 5

# Preview without writing
python3 generate_post.py --dry-run

# List all available topics
python3 generate_post.py --list
```
